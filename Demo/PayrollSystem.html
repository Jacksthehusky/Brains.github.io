Payroll System

Manage your organisation's employees and all facets of their salaries

Multi-currency Real-time Reports Multi-company Secure

System Overview

Built with Visual Studio .NET Framework and SQL Server Database
This comprehensive payroll module handles all employee data, multi-currency payslip generation, and daily attendance tracking. It automates key calculations for taxes, social security (NSSF), and end-of-service, while managing daily exchange rates and generating mandatory government reports
