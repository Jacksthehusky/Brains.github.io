Required System

The system software is developed under Visual Studio .NET Framework Using Sql Server Database.
Please note that the following description of the software system is based on different modules.

All systems have certain number of features of which we may list the following:

  • Management facilities in data entry screens such as creating, deleting, modifying and searching for entries.
  • All screens have the feature of using the mouse or the arrows and the ability to open multiple screens at one time.
  • Reports can be produced on printer and screen as well.

Brief Description of the Software:
This section describes in brief the features of the software.

General Specifications:

Integrated.
Multi company.
Multi currency.
User friendly.
Use of arrow keys to scroll and select options.
Fully menu driven system.
High level of security.

Access Control:

  • Protection and restriction to the access of data and operations.
  • Unlimited number of users.
  • Access restriction according to user's category.
  • Password modification is user available.

General Accounting:

  • Multiple currency accounts.
  • Journal voucher.
  • Payable and receivable accounts.
  • Trial Balance.
  • General ledger.
  • Balance Sheet.
  • Profit and Loss statement.
  • Consolidation of companies.
  • Detailed cost analysis.
  • Arabic, french, and english versions.
  • Depreciation of accounts.
  • Allows reviewing and modifying existing vouchers, and updates balances accordingly.
  • Complete automatic closing of accounting periods: generation of differences of exchange, according to Central Bank closing rate for local currency; closing of expenses & revenues accounts. Automatic generation of depreciated accounts at the end of period closing. Automatic closing a new after modification of any accounting vouchers belonging to the period.
  • Prints all usual accounting reports (Journal, Ledgers, Balance, etc.) in transaction currencies, along with fiscal and evaluation amounts.

Data Entry


Watch the quick walkthrough below to see the seamless functionality and user-friendly interface of our software in action.
  • CURRENCY Module:
    This module allows you to input the codes of the different currencies you may deal with, their descriptions as well as their symbols.
  • CHAPTERS Module:
    This module creates the codes and descriptions of the chapters that are officially recognized by the government.
  • SUB-CHAPTERS Module:
    This module manages the data of the sub-chapters account.
  • GENERAL LEDGER Module:
    This module is related to the previous two modules. It manages the data related to the general ledger.
  • AUXILIARY ACCOUNTS Module:
    This module shows the total account numbers by adding auxiliary codes to the account.
  • VOUCHER TYPE Module:
    This module inputs the different types of voucher, bank voucher, journal voucher, credit note, debit note etc...
    The above six modules form the basic programs for the accounting system. Without them, the system will be paralyzed.
  • Change Auxiliary Account Number:
    This module allows the changement of any auxiliary account number in all entry voucher automatically.

OPERATIONS Menu


Watch the quick walkthrough below to see the seamless functionality and user-friendly interface of our software in action.
  • ENTRY VOUCHER Module:
    • This module allows you to input the daily transactions that take place in the company such as payment, receipt and journal vouchers.
  • CURRENCY RATES Module:
    • This module is needed to enter the currency rates that correspond to the amount of transactions. These rates are entered on daily basis.
  • DIFFERENCE OF EXCHANGE Module:
    • This module is developed to be used at the end of the year. Its objective is to calculate the differences of values between the original value and the current value that were caused by the changes of the currency rates throughout the year on auxiliary account level.
  • CLOSING PROFIT & LOSS Module:
    • This module normally generates at the end of the fiscal year the profit and/or the loss voucher. An important feature in this module is that you can close your accounts at any date in the year.
  • ASSETS DEPRECIATION Module:
    • This module will automatically create vouchers for every depreciable asset based on percentage and lifetime formulas.
  • Closing End Of Year:
    • This module will automatically close current year and open a new one. This module could be used at any time to close the current account year.
  • INTEREST:
    • This module allows to input the interest rate given from the bank at a certain date.(Positive or Negative)
  • Renumeration of Vouchers :
    This module allows to renumerate all vouchers according to date.

REPORTS Menu

All the following modules can be produced on either the screen or the printer.

  • STATEMENT OF ACCOUNT Module:
    This module lists the transactions related to a specific account. The report may be produced from account to account and from date to date.
  • DAILY JOURNAL Module:
    The daily transactions that take place are produced in a daily journal report. Such reports are sent later to the government.
  • TRIAL BALANCE Module:
    This module produces a trial balance by currency, date, moved account, chapters, sub-chapters, and ledgers. An additional used feature is the 'Up-to-Date' option.
  • STATEMENT OF ACCOUNT BY REFERENCE Module:
    This module lists the transactions of different accounts related to a specific reference.
  • CHART OF ACCOUNT MODULE:
    This module lists all the accounts, chapters, subchapters, general ledgers, and auxiliary accounts.
  • CURRENCY RATES/CODE Module:
    This module prints out the currency rates average throughout the year.
  • INCOME STATEMENT Module:
    This module prints out the income statement in accordance with the lebanese accounting rules and regulations. You can produce income statement from a specific date to another.
  • BALANCE SHEET Module:
    This module prints out the balance sheet.
  • INTEREST Module:
    This module prints the interest value of the bank positive or negative.

END OF SESSION Menu

  • PACK & REINDEX Feature:
    This feature reorganizes the data files when requested. It is very important to use this feature once every a certain period of time.
  • BACKUP Feature:
    It is very important to store data on tapes for later use in case of probable damage. You can run this command directly from the system without quitting.

SYSTEM Menu

  • CHANGE COMPANY Module:
    By using this module, you can shift from one company to another.
    This applies on a company that has under its wings many other companies. All corresponding files will be open to the new entering data.
  • Other Features:
    The accounting system provides you with the following additional features:
    You may change the system password by using password option.
    You may choose to list auxiliary accounts by code or by description.
    You may run your system in English, Arabic, or French.
    You may number the pages of the daily journal before sending them to the government for stamping.
    You may use calculator and calendar directly from the program without quitting.
